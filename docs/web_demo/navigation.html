导航菜单

1. 水平导航

水平导航菜单适用于网站顶部导航，目前只支持到二级菜单，需要引用.navigation .navigation-menu两个样式，.navigation-menu 代表是水平导航菜单，如果需要设置当前选中的样式，则需要在 li元素中引用.current样式，二级菜单则需要在dl中引用.navigation-child样式

2. 垂直/侧边导航

垂直/侧边导航菜单适用于网站左侧树形菜单导航，目前只支持到二级菜单，需要引用.navigation .navigation-tree两个样式，.navigation-tree 代表是垂直/侧边导航菜单，如果需要设置当前选中并展开子菜单的样式，则需要在 li元素中引用.expand样式，二级菜单则需要在dl中引用.navigation-child样式
