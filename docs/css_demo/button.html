1.主题

按钮效果 按钮class 描述
btn btn-default 默认按钮样式
btn btn-primary 主要按钮，用于大多数页面上的功能按钮
btn btn-success 表示成功或积极的动作
btn btn-info 表示提示性的动作
btn btn-warning 提醒应该谨慎采取这个动作
btn btn-danger 表示这个动作危险或存在危险
btn btn-link 简化一个按钮, 使它看起来像一个链接，同时保持按钮的行为
btn 禁用按钮,禁止点击事件，只需要引用btn样式后添加disabled属性即可
