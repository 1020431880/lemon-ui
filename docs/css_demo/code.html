代码

使用了 google prettify.js 来实现代码的高亮显示。prettify 非常小巧且配置简单，使用它来实现代码的高亮显示是个不错的选择，使用十分简单。
write your code

1.HTML代码

使用代码样式模块需要引用.lemonui-code的样式，如果需要在左侧显示行号，需要添加属性linenums="true"，如果不需要行号则linenums="false"或者不写
<pre class="lemonui-code" linenums="true">write your code</pre>

2.JS代码

prettify.js 依赖jquery库。所以需要同时引入 jQuery 文件和 prettify.js 文件
<script type="text/javascript" src="lib/jquery/jquery.min.js"></script>
<script type="text/javascript" src="lib/prettify/prettify.js"></script>
