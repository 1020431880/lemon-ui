2.背景灰色极简颜色

灰色系代表极简，因为这是一种神奇的颜色，几乎可以与任何元素搭配，不易形成视觉疲劳，且永远不会过时，他们一般用于背景、边框等。比如表格头背景用#F2F2F2，鼠标划过颜色用#F5F5F5，鼠标已经划过后表格颜色用#FBFBFB..
